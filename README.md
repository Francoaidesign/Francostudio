# FRANCO Design Studio - ChatGPT Image Integration

## التشغيل
1. ارفع المشروع على Vercel.
2. من Vercel Dashboard افتح:
   Settings > Environment Variables
3. أضف:
   OPENAI_API_KEY
4. اعمل Deploy.
5. افتح الموقع واضغط "توليد الصورة".

## مهم
لا تضع مفتاح OpenAI داخل `index.html`.
المفتاح يجب أن يبقى في Backend فقط داخل Environment Variables.
